1. Ciclos

Los ciclos son estructuras que permiten repetir un bloque de instrucciones varias veces sin necesidad de escribir el mismo código una y otra vez. Son muy útiles cuando se necesita ejecutar tareas repetitivas de forma automática.

En Java existen tres tipos principales de ciclos: for, while y do-while. Cada uno se utiliza en situaciones diferentes según las necesidades del programa.

Ciclo For

El ciclo for se utiliza cuando conocemos exactamente cuántas veces queremos repetir una acción.

Sintaxis

for(inicializacion; condicion; incremento){
    instrucciones;
}

El ciclo for tiene tres partes:

  • Inicialización: crea una variable de control.
  • Condición: determina cuándo termina el ciclo.
  • Incremento: modifica la variable en cada repetición.

Ejemplo

for(int i = 10; i >= 1; i--){
    System.out.println(i);
}

Este programa mostrará los números del 10 al 1 en orden descendente.

Ciclo While

El ciclo while se utiliza cuando no sabemos exactamente cuántas veces se repetirá una acción. El ciclo continúa ejecutándose mientras la condición sea verdadera.

Sintaxis

while(condicion){
    instrucciones;
}

Ejemplo

int contador = 100;

while(contador >= 10){
    System.out.println(contador);
    contador -= 10;
}

Este programa imprime los múltiplos de 10 desde 100 hasta 10, reduciendo el contador en cada repetición.

Ciclo Do-While

El ciclo do-while es similar al ciclo while, pero tiene la particularidad de ejecutar las instrucciones al menos una vez antes de evaluar la condición.

Sintaxis

do{
    instrucciones;
}while(condicion);

Ejemplo

int valor = 5;

do{
    System.out.println("El valor es: " + valor);
    valor += 5;
}while(valor <= 25);

En este caso, el bloque de instrucciones se ejecuta primero y luego se verifica la condición, mostrando los múltiplos de 5 hasta 25.

Resumen de Ciclos

  • For: se utiliza cuando conocemos el número de repeticiones.
  • While: se utiliza cuando las repeticiones dependen de una condición.
  • Do-While: garantiza al menos una ejecución del bloque de código.

2. Métodos

Los métodos son bloques de código que permiten organizar y reutilizar instrucciones dentro de un programa. Gracias a ellos evitamos repetir código y hacemos que los programas sean más fáciles de leer y mantener.

Un método puede recibir datos llamados parámetros y también puede devolver un resultado después de ejecutar determinadas operaciones.

¿Qué es un método?

Imagina que trabajas en una panadería y todos los días debes seguir los mismos pasos para preparar pan. En lugar de recordar cada paso una y otra vez, podrías escribir una receta y simplemente seguirla cuando sea necesario.

En programación sucede algo parecido. Un método funciona como una receta: contiene instrucciones que pueden ejecutarse cada vez que las necesitemos sin tener que volver a escribirlas.

Partes de un método

public static void nombreDelMetodo(parametros){
    // instrucciones
}
Parte Significado
public El método puede ser utilizado desde cualquier parte del programa.
static No necesitamos crear un objeto para llamar al método.
void El método no devuelve ningún valor.
nombreDelMetodo Es el nombre que utilizamos para identificar y llamar al método.
parametros Son los datos que recibe el método para realizar una tarea.

Tipos de métodos

Método sin parámetros y sin retorno

Es el tipo más sencillo. Solo ejecuta instrucciones cuando lo llamamos, no necesita datos externos y no devuelve ningún resultado. 

public class MetodoMensaje {

    public static void mostrarFecha(){
        System.out.println("Hoy es lunes.");
        System.out.println("Es un buen día para aprender Java.");
    }

    public static void main(String[] args){
        mostrarFecha();

        System.out.println("El método terminó, continuamos aquí.");

        mostrarFecha();
    }
}

Este tipo de método es útil cuando siempre queremos ejecutar las mismas instrucciones sin necesidad de recibir información adicional.

Método con parámetros

Los parámetros son datos que enviamos al método para que pueda trabajar con ellos. Son parecidos a los ingredientes que utilizamos en una receta. 

public class MetodoResta {

    public static void restar(int a, int b){
        int resultado = a - b;
        System.out.println(a + " - " + b + " = " + resultado);
    }

    public static void main(String[] args){
        restar(10, 3);
        restar(50, 25);
        restar(200, 75);
    }
}

Los parámetros son las variables declaradas en el método (a y b), mientras que los argumentos son los valores reales enviados al llamarlo.

Método con retorno

A veces necesitamos que el método devuelva un resultado para utilizarlo más adelante. Para ello se utiliza la palabra return. 

public class MetodoRetorno {

    public static double calcularPromedio(double nota1, double nota2){
        double promedio = (nota1 + nota2) / 2;
        return promedio;
    }

    public static void main(String[] args){

        double resultado1 = calcularPromedio(8.0, 6.0);
        double resultado2 = calcularPromedio(9.5, 7.5);

        System.out.println("El promedio es: " + resultado1);
        System.out.println("El promedio es: " + resultado2);

        System.out.println("Otro promedio: " + calcularPromedio(10.0, 5.0));
    }
}

Este tipo de método es muy útil cuando necesitamos obtener un valor para seguir trabajando con él dentro del programa.

Resumen: tipos de métodos

Tipo void o tipo Tiene return Ejemplo
Sin parámetros, sin retorno void No public static void mostrarFecha()
Con parámetros, sin retorno void No public static void restar(int a, int b)
Con parámetros y retorno int, double, String, etc. Sí public static double promedio(double n1, double n2)

3. Vectores y Matrices

A medida que los programas crecen, surge la necesidad de almacenar grandes cantidades de información. Para resolver este problema existen los vectores y las matrices, estructuras que permiten organizar múltiples datos de forma sencilla y eficiente.

¿Qué problema resuelven?

Imagina que necesitas guardar las calificaciones de 30 estudiantes. Sin vectores tendrías que crear una variable para cada nota: 

int nota1 = 8;
int nota2 = 7;
int nota3 = 9;
// ...

Esto sería difícil de mantener. Con un vector podemos almacenar todos esos datos utilizando una sola variable. 

int[] notas = {8, 7, 9, 6, 10};

¿Qué es un vector?

Un vector (también conocido como arreglo o array) es una estructura que almacena varios valores del mismo tipo bajo un único nombre.

Cada elemento ocupa una posición llamada índice, y los índices comienzan desde 0.

Índice 0 1 2 3 4
Valor 8 7 9 6 10

Para acceder al primer elemento utilizamos: 

notas[0]

Para acceder al tercer elemento: 

notas[2]

Crear un vector

Existen diferentes formas de crear un vector en Java.

Forma 1: Inicialización directa

int[] numeros = {10, 20, 30, 40, 50};

Forma 2: Definiendo el tamaño

int[] numeros = new int[5];

numeros[0] = 10;
numeros[1] = 20;
numeros[2] = 30;
numeros[3] = 40;
numeros[4] = 50;

Ambas formas producen el mismo resultado.

Ejemplo completo de vector

public class VectorBasico {

    public static void main(String[] args) {

        String[] ciudades = {
            "Quito",
            "Guayaquil",
            "Cuenca",
            "Ambato",
            "Loja"
        };

        System.out.println(ciudades[0]);
        System.out.println(ciudades[2]);
        System.out.println(ciudades[4]);

        System.out.println(ciudades.length);
    }
}

¿Cómo funciona paso a paso?

La computadora realiza las siguientes acciones:

  1. Crea un vector con cinco elementos.
  2. Guarda cada ciudad en una posición diferente.
  3. Accede a los elementos utilizando índices.
  4. Muestra el tamaño total utilizando length.

Recorrer un vector

Normalmente utilizamos un ciclo for para recorrer todos los elementos de un vector. 

double[] temperaturas = {18.5, 22.0, 19.3, 25.1, 17.8};

for(int i = 0; i < temperaturas.length; i++){
    System.out.println(temperaturas[i]);
}

La variable i representa cada posición del vector.

¿Qué es una matriz?

Si un vector es una fila de datos, una matriz es una tabla compuesta por filas y columnas.

Una matriz puede compararse con una hoja de Excel donde cada celda tiene una fila y una columna.

Columna 0 Columna 1 Columna 2
Fila 0 1 2 3
Fila 1 4 5 6
Fila 2 7 8 9

Para acceder al número 5 utilizamos: 

matriz[1][1]

Ejemplo completo de matriz

int[][] notas = {
    {10, 8, 9},
    {7, 6, 8},
    {9, 10, 7}
};

System.out.println(notas[0][0]);
System.out.println(notas[1][2]);

Resultado: 

10
8

¿Cómo funciona paso a paso?

  1. Se crea una matriz de 3 filas y 3 columnas.
  2. Cada número ocupa una posición específica.
  3. El primer índice representa la fila.
  4. El segundo índice representa la columna.
  5. notas[0][0] accede al valor 10 y notas[1][2] accede al valor 8.

4. Listas (ArrayList)

En muchos programas necesitamos almacenar una cantidad de datos que puede cambiar constantemente. Para estos casos Java ofrece las listas dinámicas mediante la clase ArrayList.

La limitación de los vectores

Los vectores tienen un problema importante: su tamaño es fijo. Una vez que declaramos un vector, no podemos agregar más elementos de los que fueron definidos inicialmente. 

int[] numeros = new int[5];

El vector anterior siempre tendrá exactamente 5 posiciones. Si más adelante necesitamos una sexta posición, no podremos agregarla fácilmente.

En muchos programas reales no sabemos cuántos datos necesitaremos almacenar. Para resolver este problema utilizamos listas dinámicas.

¿Qué es un ArrayList?

Un ArrayList es una colección que puede crecer o disminuir automáticamente según sea necesario. Java se encarga de administrar el tamaño internamente, permitiendo agregar o eliminar elementos con facilidad.

Vector vs ArrayList

Característica Vector (int[]) ArrayList
Tamaño Fijo, definido al crear Dinámico, cambia durante la ejecución
Agregar elementos No se puede superar el tamaño inicial lista.add(elemento)
Eliminar elementos No se puede reducir lista.remove(indice)
Acceso por índice vector[i] lista.get(i)

Sintaxis básica

Antes de utilizar un ArrayList debemos importar la clase correspondiente: 

import java.util.ArrayList;

Luego creamos la lista indicando el tipo de dato que almacenará: 

ArrayList<String> nombres = new ArrayList<>();
ArrayList<Integer> numeros = new ArrayList<>();
ArrayList<Double> precios = new ArrayList<>();

Métodos principales

Método ¿Qué hace? Ejemplo
add(elemento) Agrega un elemento al final lista.add("Ana")
get(i) Obtiene el elemento en la posición indicada lista.get(0)
remove(i) Elimina el elemento de una posición lista.remove(2)
size() Devuelve el número de elementos lista.size()
contains(x) Indica si un elemento existe lista.contains("Ana")
clear() Elimina todos los elementos lista.clear()

Ejemplo completo: Lista básica

import java.util.ArrayList;

public class ListaBasica {

    public static void main(String[] args) {

        ArrayList<String> materias = new ArrayList<>();

        materias.add("Matemáticas");
        materias.add("Programación");
        materias.add("Física");
        materias.add("Inglés");

        System.out.println("Número de materias: " + materias.size());

        System.out.println("Primera materia: " + materias.get(0));
        System.out.println("Tercera materia: " + materias.get(2));
    }
}

¿Cómo funciona paso a paso?

  1. Se crea una lista vacía.
  2. Se agregan elementos mediante add().
  3. La lista aumenta su tamaño automáticamente.
  4. size() devuelve la cantidad de elementos.
  5. get() permite acceder a una posición específica.

Ejemplo: Agregar y eliminar elementos

ArrayList<String> animales = new ArrayList<>();

animales.add("Perro");
animales.add("Gato");
animales.add("Conejo");
animales.add("Loro");

System.out.println(animales);

Cuando se elimina un elemento, los elementos posteriores se desplazan automáticamente para ocupar el espacio vacío.

Ejemplo: Recorrer una lista

ArrayList<Integer> calificaciones = new ArrayList<>();

calificaciones.add(9);
calificaciones.add(7);
calificaciones.add(10);

for(int nota : calificaciones){
    System.out.println("Nota: " + nota);
}

5. Manejo de Archivos

Hasta ahora hemos trabajado con variables, vectores, matrices y listas para almacenar información. Sin embargo, todos esos datos permanecen únicamente en la memoria de la computadora mientras el programa está en ejecución.

Cuando cerramos el programa, la información almacenada desaparece. Por esta razón es necesario utilizar archivos, ya que permiten guardar datos de forma permanente para poder utilizarlos nuevamente más adelante.

¿Por qué necesitamos archivos?

Los archivos permiten almacenar información importante para que siga existiendo incluso después de cerrar el programa. Gracias a ellos es posible guardar configuraciones, registros, tareas, notas, usuarios y muchos otros tipos de información.

Algunos ejemplos donde se utiliza el manejo de archivos son:

  • Guardar información de estudiantes.
  • Registrar ventas de una tienda.
  • Almacenar listas de tareas.
  • Guardar configuraciones de una aplicación.
  • Crear reportes y documentos.

Clases necesarias

Para trabajar con archivos en Java debemos importar algunas clases del paquete java.io. 

import java.io.FileWriter;
import java.io.BufferedWriter;
import java.io.FileReader;
import java.io.BufferedReader;
import java.io.IOException;
Clase Función
FileWriter Permite escribir información en un archivo.
BufferedWriter Mejora el rendimiento al escribir.
FileReader Permite leer información desde un archivo.
BufferedReader Facilita la lectura línea por línea.
IOException Permite manejar errores relacionados con archivos.

Escribir en un archivo

El siguiente ejemplo crea un archivo llamado tutorial_java.txt y guarda información dentro de él. 

import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class GuardarInformacion {

    public static void main(String[] args) {

        String nombreArchivo = "tutorial_java.txt";

        try(BufferedWriter escritor =
            new BufferedWriter(new FileWriter(nombreArchivo))){

            escritor.write("Nombre: Jean Pierre");
            escritor.newLine();

            escritor.write("Tema: Programacion en Java");
            escritor.newLine();

            escritor.write("Nivel: Principiante");
            escritor.newLine();

            System.out.println("Archivo guardado correctamente.");

        }catch(IOException e){

            System.out.println("Error: " + e.getMessage());

        }
    }
}

¿Cómo funciona paso a paso?

  1. Se crea el archivo tutorial_java.txt.
  2. Se abre un canal de escritura.
  3. Se escriben varias líneas de texto.
  4. Se guarda la información.
  5. El archivo se cierra automáticamente.

Leer un archivo

Después de guardar información también podemos leerla. 

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class MostrarInformacion {

    public static void main(String[] args) {

        String nombreArchivo = "tutorial_java.txt";

        try(BufferedReader lector =
            new BufferedReader(new FileReader(nombreArchivo))){

            String linea;

            System.out.println("Contenido del archivo:");

            while((linea = lector.readLine()) != null){

                System.out.println(linea);

            }

        }catch(IOException e){

            System.out.println("Error: " + e.getMessage());

        }
    }
}

¿Cómo funciona paso a paso?

  1. Se abre el archivo.
  2. Se lee una línea a la vez.
  3. La información se muestra en pantalla.
  4. Cuando ya no existen más líneas, el ciclo termina.
  5. El archivo se cierra automáticamente.

Procesar información de un archivo

Además de leer archivos, también podemos analizar su contenido. En este ejemplo contamos cuántas líneas tiene un archivo. 

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class ContarLineas {

    public static void main(String[] args) {

        String nombreArchivo = "tutorial_java.txt";

        int contador = 0;

        try(BufferedReader lector =
            new BufferedReader(new FileReader(nombreArchivo))){

            while(lector.readLine() != null){

                contador++;

            }

            System.out.println(
                "Total de lineas: " + contador
            );

        }catch(IOException e){

            System.out.println("Error: " + e.getMessage());

        }
    }
}

El bloque try-catch

Cuando trabajamos con archivos pueden ocurrir errores. Por ejemplo, el archivo podría no existir o no tener permisos para ser abierto.

Para evitar que el programa se detenga inesperadamente utilizamos los bloques try-catch. 

try{

    // Codigo que puede generar errores

}catch(IOException e){

    System.out.println(
        "Error: " + e.getMessage()
    );

}

El bloque try contiene el código que puede fallar y el bloque catch se ejecuta cuando ocurre algún problema.
