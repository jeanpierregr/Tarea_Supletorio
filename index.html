1. Ciclos

Los ciclos son estructuras que permiten repetir un bloque de instrucciones varias veces sin necesidad de escribir el mismo código una y otra vez. Son muy útiles cuando se necesita ejecutar tareas repetitivas de forma automática.

En Java existen tres tipos principales de ciclos: for, while y do-while. Cada uno se utiliza en situaciones diferentes según las necesidades del programa.

Ciclo For

El ciclo for se utiliza cuando conocemos exactamente cuántas veces queremos repetir una acción.

Sintaxis

for(inicializacion; condicion; incremento){
    instrucciones;
}

El ciclo for tiene tres partes:

  • Inicialización: crea una variable de control.
  • Condición: determina cuándo termina el ciclo.
  • Incremento: modifica la variable en cada repetición.

Ejemplo

for(int i = 1; i <= 5; i++){
    System.out.println(i);
}

Este programa mostrará los números del 1 al 5 en pantalla.

Ciclo While

El ciclo while se utiliza cuando no sabemos exactamente cuántas veces se repetirá una acción. El ciclo continúa ejecutándose mientras la condición sea verdadera.

Sintaxis

while(condicion){
    instrucciones;
}

Ejemplo

int numero = 1;

while(numero <= 5){
    System.out.println(numero);
    numero++;
}

Este programa imprime los números del 1 al 5 utilizando una condición que se evalúa antes de cada repetición.

Ciclo Do-While

El ciclo do-while es similar al ciclo while, pero tiene la particularidad de ejecutar las instrucciones al menos una vez antes de evaluar la condición.

Sintaxis

do{
    instrucciones;
}while(condicion);

Ejemplo

int numero = 1;

do{
    System.out.println(numero);
    numero++;
}while(numero <= 5);

En este caso, el bloque de instrucciones se ejecuta primero y luego se verifica la condición.

Resumen de Ciclos

  • For: se utiliza cuando conocemos el número de repeticiones.
  • While: se utiliza cuando las repeticiones dependen de una condición.
  • Do-While: garantiza al menos una ejecución del bloque de código.

2. Métodos

Los métodos son bloques de código que permiten organizar y reutilizar instrucciones dentro de un programa. Gracias a ellos evitamos repetir código y hacemos que los programas sean más fáciles de leer y mantener.

Un método puede recibir datos llamados parámetros y también puede devolver un resultado después de ejecutar determinadas operaciones.

¿Qué es un método?

Imagina que trabajas en una panadería y todos los días debes seguir los mismos pasos para preparar pan. En lugar de recordar cada paso una y otra vez, podrías escribir una receta y simplemente seguirla cuando sea necesario.

En programación sucede algo parecido. Un método funciona como una receta: contiene instrucciones que pueden ejecutarse cada vez que las necesitemos sin tener que volver a escribirlas.

Partes de un método

public static void nombreDelMetodo(parametros){
    // instrucciones
}
Parte Significado
public El método puede ser utilizado desde cualquier parte del programa.
static No necesitamos crear un objeto para llamar al método.
void El método no devuelve ningún valor.
nombreDelMetodo Es el nombre que utilizamos para identificar y llamar al método.
parametros Son los datos que recibe el método para realizar una tarea.

Tipos de métodos

Método sin parámetros y sin retorno

Es el tipo más sencillo. Solo ejecuta instrucciones cuando lo llamamos, no necesita datos externos y no devuelve ningún resultado. 

public class MetodoSaludar {

    public static void saludar(){
        System.out.println("¡Hola! Bienvenido al programa.");
        System.out.println("Este es tu primer método en Java.");
    }

    public static void main(String[] args){
        saludar();

        System.out.println("El método terminó, continuamos aquí.");

        saludar();
    }
}

Este tipo de método es útil cuando siempre queremos ejecutar las mismas instrucciones sin necesidad de recibir información adicional.

Método con parámetros

Los parámetros son datos que enviamos al método para que pueda trabajar con ellos. Son parecidos a los ingredientes que utilizamos en una receta. 

public class MetodoSuma {

    public static void sumar(int a, int b){
        int resultado = a + b;
        System.out.println(a + " + " + b + " = " + resultado);
    }

    public static void main(String[] args){
        sumar(3, 7);
        sumar(15, 20);
        sumar(100, 50);
    }
}

Los parámetros son las variables declaradas en el método (a y b), mientras que los argumentos son los valores reales enviados al llamarlo.

Método con retorno

A veces necesitamos que el método devuelva un resultado para utilizarlo más adelante. Para ello se utiliza la palabra return. 

public class MetodoRetorno {

    public static int calcularCuadrado(int numero){
        int cuadrado = numero * numero;
        return cuadrado;
    }

    public static void main(String[] args){

        int resultado1 = calcularCuadrado(4);
        int resultado2 = calcularCuadrado(7);

        System.out.println("El cuadrado de 4 es: " + resultado1);
        System.out.println("El cuadrado de 7 es: " + resultado2);

        System.out.println("El cuadrado de 10 es: " + calcularCuadrado(10));
    }
}

Este tipo de método es muy útil cuando necesitamos obtener un valor para seguir trabajando con él dentro del programa.

Resumen: tipos de métodos

Tipo void o tipo Tiene return Ejemplo
Sin parámetros, sin retorno void No public static void saludar()
Con parámetros, sin retorno void No public static void sumar(int a, int b)
Con parámetros y retorno int, double, String, etc. Sí public static int cuadrado(int n)

3. Vectores y Matrices

A medida que los programas crecen, surge la necesidad de almacenar grandes cantidades de información. Para resolver este problema existen los vectores y las matrices, estructuras que permiten organizar múltiples datos de forma sencilla y eficiente.

¿Qué problema resuelven?

Imagina que necesitas guardar las calificaciones de 30 estudiantes. Sin vectores tendrías que crear una variable para cada nota: 

int nota1 = 8;
int nota2 = 7;
int nota3 = 9;
// ...

Esto sería difícil de mantener. Con un vector podemos almacenar todos esos datos utilizando una sola variable. 

int[] notas = {8, 7, 9, 6, 10};

¿Qué es un vector?

Un vector (también conocido como arreglo o array) es una estructura que almacena varios valores del mismo tipo bajo un único nombre.

Cada elemento ocupa una posición llamada índice, y los índices comienzan desde 0.

Índice 0 1 2 3 4
Valor 8 7 9 6 10

Para acceder al primer elemento utilizamos: 

notas[0]

Para acceder al tercer elemento: 

notas[2]

Crear un vector

Existen diferentes formas de crear un vector en Java.

Forma 1: Inicialización directa

int[] numeros = {10, 20, 30, 40, 50};

Forma 2: Definiendo el tamaño

int[] numeros = new int[5];

numeros[0] = 10;
numeros[1] = 20;
numeros[2] = 30;
numeros[3] = 40;
numeros[4] = 50;

Ambas formas producen el mismo resultado.

Ejemplo completo de vector

public class VectorBasico {

    public static void main(String[] args) {

        String[] frutas = {
            "Manzana",
            "Banana",
            "Cereza",
            "Durazno",
            "Uva"
        };

        System.out.println(frutas[0]);
        System.out.println(frutas[2]);
        System.out.println(frutas[4]);

        System.out.println(frutas.length);
    }
}

¿Cómo funciona paso a paso?

La computadora realiza las siguientes acciones:

  1. Crea un vector con cinco elementos.
  2. Guarda cada fruta en una posición diferente.
  3. Accede a los elementos utilizando índices.
  4. Muestra el tamaño total utilizando length.

Recorrer un vector

Normalmente utilizamos un ciclo for para recorrer todos los elementos de un vector. 

int[] numeros = {15, 30, 45, 60, 75};

for(int i = 0; i < numeros.length; i++){
    System.out.println(numeros[i]);
}

La variable i representa cada posición del vector.

¿Qué es una matriz?

Si un vector es una fila de datos, una matriz es una tabla compuesta por filas y columnas.

Una matriz puede compararse con una hoja de Excel donde cada celda tiene una fila y una columna.

Columna 0 Columna 1 Columna 2
Fila 0 1 2 3
Fila 1 4 5 6
Fila 2 7 8 9

Para acceder al número 5 utilizamos: 

matriz[1][1]

Ejemplo completo de matriz

int[][] matriz = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

System.out.println(matriz[1][1]);

Resultado: 

5

¿Cómo funciona paso a paso?

  1. Se crea una matriz de 3 filas y 3 columnas.
  2. Cada número ocupa una posición específica.
  3. El primer índice representa la fila.
  4. El segundo índice representa la columna.
  5. matriz[1][1] accede al valor 5.

Los vectores y matrices permiten almacenar grandes cantidades de información de forma organizada. Gracias a estas estructuras es posible trabajar con colecciones de datos sin necesidad de crear decenas de variables independientes.
